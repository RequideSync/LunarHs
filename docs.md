## 🌟 Features

- Upload keys with one button
- Customizable options:
  - Key system name
  - Key expiration time
  - Key prefix (e.g., `Test_5104`)
  - Edit password
  - Redirect URL
  - GET key page URL
- Auto-generated verification API
- API Docs auto-generated
- API copy button included
- System edit & delete support
